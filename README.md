# Mémoire+

**Un journal d'activités accessible, simple et puissant pour tous**

Mémoire+ est une application web gratuite et open source qui permet de documenter ses activités quotidiennes avec date, horaires, photos et descriptions détaillées. Idéal pour garder une trace de vos actions, prouver votre engagement, ou simplement vous souvenir.



## Fonctionnalités

### Organisation par dossiers
- Créez plusieurs dossiers pour différents usages (militant, professionnel, personnel, tâches ménagères...)
- Chaque dossier a ses propres types d'activités personnalisables.

### Enregistrement complet
- **Date** : choisissez n'importe quelle date (pratique pour rattraper des oublis)
- **Type d'activité** : créez vos propres catégories
- **Horaires** : heure de début et de fin avec calcul automatique de la durée
- **Description** : texte libre pour tous les détails
- **Photo** : ajoutez une image pour illustrer votre activité.

### Export et statistiques
- Téléchargez vos données en format HTML tableau
- Filtrez par dossier (un seul ou tous)
- Filtrez par type d'activité
- Statistiques automatiques : nombre d'activités et temps total
- Tableaux clairs, imprimables et partageables.

### Accessibilité
- Barre d'outils d'accessibilité intégrée ([Faites Mieux](https://github.com/alixb-accessible/faites-mieux-toolbar))
- Police Lexend optimisée pour la lisibilité
- Interface intuitive et responsive (mobile/tablette/ordinateur)
- Design inclusif et coloré.

### Sauvegarde locale
- Toutes vos données restent **sur votre appareil**
- Aucun compte nécessaire, aucun serveur distant
- Persistance : vos données ne sont jamais perdues
- Confidentialité totale.



## Cas d'usage

### Pour la mémoire
- Aide-mémoire pour personnes ayant des troubles de la mémoire
- Journal de vie pour se souvenir de ses journées
- Carnet de bord thérapeutique.

### Pour la preuve
- Documentation d'activités professionnelles, militantes ou syndicales
- Suivi de tâches ménagères (pour éducateurs, tuteurs, etc.)
- Historique d'activités professionnelles
- Preuve de présence et d'engagement.

### Pour l'organisation
- Suivi de projets personnels
- Gestion du temps et productivité
- Journal de bénévolat
- Registre d'activités sportives.


## Utilisation

### En ligne
Accédez directement à l'application : **[https://alixb-accessible.github.io/memoire-plus/](https://alixb-accessible.github.io/memoire-plus/)**

### Installation locale
1. Téléchargez le fichier `index.html`
2. Ouvrez-le dans votre navigateur (Chrome, Firefox, Safari, Edge...)
3. Ajoutez-le à vos favoris pour un accès rapide


## Guide d'utilisation

### 1. Créer un dossier
Cliquez sur **"+ Nouveau dossier"** et donnez-lui un nom (ex: "Activités syndicales", "Tâches ménagères", "Bénévolat"...)

### 2. Créer des types d'activités
Cliquez sur le **"+"** à côté du menu déroulant d'activités pour créer vos propres catégories (ex: "Réunion", "Manifestation", "Ménage cuisine"...)

### 3. Enregistrer une activité
- Choisissez la date (par défaut : aujourd'hui)
- Sélectionnez le type d'activité
- Ajoutez les horaires (optionnel mais recommandé)
- Décrivez votre activité
- Ajoutez une photo si besoin
- Cliquez sur **"Enregistrer l'activité"**

### 4. Télécharger vos données
Dans la section jaune **"Télécharger mes données"** :
- Choisissez le dossier (actuel, tous, ou un spécifique)
- Choisissez le type d'activité (toutes ou une spécifique)
- Cliquez sur **"Télécharger"**

Vous obtiendrez un fichier HTML avec un tableau professionnel incluant :
- Toutes vos activités filtrées
- Les photos (réduites pour l'impression)
- Les statistiques (nombre d'activités et temps total)
- Un document imprimable et partageable


## Design

L'application utilise un dégradé diagonal aux couleurs du **drapeau des fiertés handicapées** (Disability Pride Flag), créé par Ann Magill en 2019 et révisé en 2021 pour améliorer l'accessibilité visuelle.

Les couleurs représentent :
- **Gris foncé** : le deuil et la révolte pour les personnes handicapées tuées par négligence, par la violence capacitiste et eugéniste
- **Vert** : la sensorialité
- **Bleu** : l'émotionnel et le psychiatrique
- **Blanc** : le handicap invisible et non diagnostiqué
- **Jaune** : le cognitif et l'intellectuel
- **Rouge** : le physique

Police **Lexend** : conçue pour améliorer la vitesse de lecture et réduire la fatigue visuelle.

- Navigateur moderne avec JavaScript activé

## Confidentialité et sécurité

- **Aucune donnée n'est envoyée sur internet**
- Tout est stocké localement dans votre navigateur
- Aucun cookie, aucun tracker, aucune analytique
- Aucun compte nécessaire
- Vos photos restent sur votre appareil

**Important** : Si vous videz le cache de votre navigateur ou utilisez la navigation privée, vos données seront perdues. Pensez à télécharger régulièrement vos exports.


## Sauvegarde de vos données

Pour ne pas perdre vos données :
1. Téléchargez régulièrement vos exports HTML (ils contiennent tout)
2. Marquez la page en favori pour la retrouver facilement
3. Ne supprimez pas les données de navigation du site
